# Azure AI Foundry Model Deployment Lab

## Project Overview

This project demonstrates the end-to-end process of creating and managing an AI solution using Microsoft Azure AI Foundry.

The lab covers:

- Creating an Azure AI Foundry Project
- Exploring Azure AI Foundry Portal
- Deploying a GPT Model
- Testing Models in the Playground
- Connecting Applications to Azure AI Foundry
- Speech Recognition and Speech Synthesis
- Computer Vision Capabilities
- Information Extraction from Images
- Responsible AI and Safety Guardrails
